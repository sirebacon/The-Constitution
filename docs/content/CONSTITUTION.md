## Core Design Principles

These principles guide every drafting decision. When in doubt, refer back here.

1. **Majority rule with minority protection** — majorities govern, but cannot strip fundamental rights from minorities.
2. **No permanent minority veto** — structural features must not allow a permanent political minority to block majority governance indefinitely.
3. **Resilience against backsliding** — the constitution must be hard to dismantle from within by bad-faith actors.
4. **Adaptability** — legitimate change must be possible without requiring near-unanimity.
5. **Accountability** — power must be traceable, visible, and removable.
6. **Long-term thinking** — governance structures must be capable of addressing multi-generational problems.

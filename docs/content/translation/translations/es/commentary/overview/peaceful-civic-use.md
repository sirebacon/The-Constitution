# Uso Cívico Pacífico

## Propósito del Proyecto

Este proyecto existe para la discusión cívica pacífica, la redacción constitucional, la educación, la traducción, la crítica, la simulación y la defensa democrática.

Es un proyecto público de referencia constitucional. No es una milicia, una organización operativa ni un vehículo para la coacción o la fuerza.

## Lo Que el Proyecto No Respalda

Este proyecto no respalda:

- la violencia
- las amenazas
- el sabotaje
- la planificación operativa para la coacción o la fuerza
- la acción ilícita ni la incitación a una acción ilícita inminente

El material enviado con esos fines queda fuera del proyecto y puede ser eliminado.

## Cómo Utilizar el Proyecto

Las personas son libres de:

- leer y criticar el borrador
- revisarlo o bifurcarlo
- traducirlo
- enseñarlo
- organizar debates lícitos en torno a él
- hacer campaña a favor de partes de él si les resultan persuasivas

El proyecto está pensado para que otros puedan utilizarlo sin necesidad de que una sola figura pública se ponga al frente.

## Posición de las Personas Mantenedoras

Las personas mantenedoras pueden eliminar contenido o cerrar debates que intenten convertir el proyecto en una defensa de la fuerza, de la acción ilícita o de la coordinación operativa.

El propósito del repositorio y del sitio web es la discusión constitucional y la redacción pública, no la coacción.

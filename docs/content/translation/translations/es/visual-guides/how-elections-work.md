# Cómo Funcionan Las Elecciones

Esta guía visual explica qué es diferente en las elecciones federales bajo esta constitución: el método de votación, cómo funciona el registro, qué protecciones evitan la manipulación y cómo el electorado puede revocar a funcionarios.

## Cómo Leer Esta Guía

Cada tarjeta cubre una característica del sistema electoral. Usa los botones de categoría para centrarte en métodos de votación, acceso y registro, protecciones de integridad o el mecanismo de revocación.

## Qué Cubre Esta Guía

- voto preferencial y distritos plurinominales para la Cámara
- voto por eliminación instantánea para la Asamblea Regional
- registro automático de votantes
- protección de la fecha electoral frente a manipulación de emergencia
- redistribución independiente con reglas anti-gerrymandering
- revocación de miembros de la Cámara y de la Asamblea Regional

## Qué No Hace Esta Guía

Esta guía es una capa de orientación, no el texto constitucional en sí.

Para la redacción exacta y el detalle jurídico, lee:

# Ciclo De Los Poderes De Emergencia

Esta guía visual muestra cómo comienza una emergencia nacional, cómo se limita, cómo caduca y cómo responde el sistema constitucional si el ejecutivo intenta ampliar o evadir esos límites.

## Cómo Leer Esta Guía

La guía tiene tres rutas:

- ruta constitucional ordinaria
- ruta de caducidad cuando no se concede la aprobación
- ruta de abuso cuando el poder de emergencia se estira más allá de los límites legales

Usa los botones de ruta para centrarte en una rama del sistema de emergencia a la vez.

## Qué Cubre Esta Guía

- declaración presidencial
- remisión a la Asamblea Regional dentro de 48 horas
- plazo de aprobación de 30 días
- caducidad automática si no se aprueba
- renovaciones de 30 días
- máximo de 180 días sin voto de 2/3 de ambas cámaras
- salvaguardas judiciales y de rendición de cuentas

## Qué No Hace Esta Guía

Esta guía es una capa de orientación, no el texto constitucional en sí.

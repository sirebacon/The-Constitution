# El Proceso De Enmienda

Esta guía visual explica cómo puede modificarse esta constitución, en qué se diferencian las dos vías de enmienda y qué no puede modificarse bajo ningún procedimiento.

## Cómo Leer Esta Guía

La tabla comparativa muestra la Vía 1 y la Vía 2 lado a lado. La sección del núcleo no enmendable enumera cada disposición que no puede ser enmendada, derogada ni suspendida por ninguna mayoría en ningún momento.

## Qué Cubre Esta Guía

- Vía 1 (enmiendas estructurales): voto del Congreso, preautorización y ratificación estatal
- Vía 2 (expansión de derechos): voto del Congreso, ratificación estatal y referéndum nacional
- el núcleo no enmendable: once disposiciones bloqueadas de forma permanente
- qué ocurre si se deniega la preautorización
- comparación con el proceso de enmienda del actual Artículo V de Estados Unidos

## Qué No Hace Esta Guía

Esta guía es una capa de orientación, no el texto constitucional en sí.

Para la redacción exacta y el detalle jurídico, lee:

- [Artículo XI](articles/XI-amendments.md)

# 宪法评分卡

使用本文档跟踪草案质量的演变情况。目标不仅在于对文本进行评级，更在于确定下一步改进工作的重点所在。

---

## 评分标准

每个领域按**1至10分**进行评分。

- **10分** — 优秀；条理清晰、可执行，接近定稿
- **8–9分** — 较强；仅需针对性修订
- **6–7分** — 基础良好；仍需实质性修订
- **4–5分** — 概念上有前景，但欠缺充分发展或存在不稳定性
- **1–3分** — 需要大幅重新设计

---

## 评估标准

就以下维度对每个部分进行评分：

| 标准 | 评估问题 |
|----------|-------------|
| **清晰度** | 初次阅读时规则是否易于理解？ |
| **内部一致性** | 是否与宪法其他部分契合无间？ |
| **可执行性** | 法院、委员会或政治行为者能否在实践中适用该规则？ |
| **民主正当性** | 是否在保障多数决原则的同时保护少数权利，维护问责机制？ |
| **抗滥用性** | 是否封堵了明显的恶意或威权主义漏洞？ |
| **草案规范性** | 是否足够简洁，发挥宪法文本的功能而非沦为法规汇编？ |

建议评分方法：

1. 对每项标准给出1至10分的评分。
2. 取平均值作为该部分得分。
3. 必要时精确至0.5分。
4. 记录该得分的主要理由及下一步改进措施。

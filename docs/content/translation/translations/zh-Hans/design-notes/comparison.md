# 比较：现行宪法与修订宪法

逐项对照参考，说明修改内容及其原因。

---

| 领域 | 美国现行宪法 | 修订宪法 |
|------|------------|---------|
| **选举权** | 未明确保障；通过修正案推断与保护 | 明确的宪法权利；自动登记选民 |
| **总统选举** | 选举人团制度 | 全国直接普选，采用排序复选投票制 |
| **投票制度** | 未作规定（各州默认采用相对多数制） | 联邦层面强制实行排序复选投票制 |
| **国会选区** | 单一议员、赢者通吃 | 多议员比例代表选区 |
| **选区操纵** | 无保护措施；交由各州及法院处理 | 强制实行独立算法划分选区 |
| **参议院** | 每州两名参议员，不论人口多寡 | 地区议会，按人口比例加权 |
| **冗长辩论阻挠** | 不在宪法之中；属参议院议事规则 | 明令禁止——恢复多数决原则 |
| **紧急权力** | 无时限规定（《国家紧急状态法》为法律，非宪法） | 限30天，须经国会续期 |
| **赦免权** | 范围宽泛，无明确限制 | 不得自我赦免；不得赦免共谋者 |
| **战争权力** | 规定模糊；国会宣战但总统统率军队 | 须经国会授权 |
| **最高法院规模** | 9名大法官（由法律规定） | 15名大法官（由宪法规定） |
| **大法官任期** | 终身任职 | 15年交错任期 |
| **大法官任命** | 总统提名加参议院多数通过 | 总统提名加非党派委员会审核 |
| **司法伦理** | 无宪法层面的要求 | 强制性伦理准则及执行机制 |
| **隐私权** | 由推断而来（第四修正案之延伸） | 明确的宪法权利 |
| **平等权利** | 平等保护条款（第十四修正案）；《平等权利修正案》从未获批准 | 明确保障跨种族、性别、性取向、残障等方面的平等 |
| **积极权利** | 无 | 医疗保健底线、教育权、住房保障 |
| **环境权利** | 无 | 享有清洁空气、清洁水源及气候稳定的权利 |
| **数字权利** | 无 | 明确的数据隐私权及算法权利 |
| **持枪权** | "组织良好的民兵"——存在争议 | 个人权利，受合理规制约束 |
| **反政变条款** | 无 | 永久取消资格并追究刑事责任 |
| **竞选财务** | 大部分被法院判决推翻 | 以宪法确立限额与披露要求 |
| **旋转门** | 仅有法律规定，易于修改 | 5年限制期入宪 |
| **媒体多元化** | 无宪法层面的规定 | 所有权集中度限制 |
| **修宪程序** | 单一路径：三分之二加四分之三州批准（极为困难） | 双轨制：结构性事项（路径一，较难）与权利扩展（路径二，适中） |
| **不可修改核心** | 无 | 选举权、反颠覆条款、司法独立 |
| **领土与哥伦比亚特区** | 无建州途径；无表决权代表 | 明确的建州路径；完整的代表权 |

---

## 保留内容

- 三权分立（行政、立法、司法）
- 两院制（经改革，未予废除）
- 联邦主义（各州保留保留权力）
- 全部经典公民自由（言论、宗教、新闻、集会、正当程序等）
- 总统制（未替换为议会制）
- 联邦法律在列举事项上凌驾于州法律之上
- 各分支之间的制衡机制

## 核心设计原则

这些原则指导每一项起草决策。如有疑问，请回归此处。

1. **多数统治与少数保护** — 多数决定治理，但不得剥夺少数群体的基本权利。
2. **无永久性少数否决权** — 制度性安排不得允许永久性政治少数无限期阻挠多数治理。
3. **抵御倒退的韧性** — 宪法必须难以被恶意行为者从内部瓦解。
4. **适应性** — 合法变革必须成为可能，无需近乎全体一致的同意。
5. **问责性** — 权力必须可追溯、可见、可撤销。
6. **长远思维** — 治理结构必须具备应对跨代际问题的能力。

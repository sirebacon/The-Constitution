# 第一条 注释

## 本条旨在解决的问题

现行美国宪法结构在原则上保护选举，但将民主体制的大部分内容留给党派操弄、规范不足与制度惯例处理。

本条将选举制度本身视为宪法架构，而非普通的政治战利品。

## 为何如此多的选举制度设计载入宪法

选举制度塑造代表性、政党结构、合法性，以及选民更换执政者的能力。

正因如此，本草案将以下内容宪法化：

- 投票权
- 自动登记
- 选举时间安排
- 核心选举方法
- 政党资格认定与反卡特尔规则
- 选举管理独立性

其目的不在于将每一项技术性规则永久固化，而在于将民主底线置于普通党派改写之上。

## 为何本条纳入反卡特尔与基础设施规则

民主可能在未正式废除选举的情况下遭到扭曲。

因此，本条所规制的范围不仅涵盖对选票的直接排斥，还涵盖对核心公共或受公共监管的选举基础设施的排斥。其目的在于防止程序性守门行为成为开放民主竞争的替代障碍。

## 为何本条仍有所克制

宪法不应监管每一个私人辩论主办方、公民社团或媒体平台。

本条所划定的界限是经过深思熟虑的：

- 核心公共及受公共监管的选举基础设施属于宪法底线的范畴
- 较为软性或较为私密的协调问题主要留由制定法、竞争法与选举法处理

这一划分使本条在保护民主的同时，避免演变为规范整个政治生活的一般性法典。

# 和平公民用途

## 项目宗旨

本项目旨在服务于和平的公民讨论、宪法起草、教育、翻译、批评、模拟测试以及民主倡议。

本项目是一个公开的宪法参考项目，不是民兵组织、行动性组织，也不是用于胁迫或使用武力的工具。

## 本项目不支持的内容

本项目不支持：

- 暴力
- 威胁
- 破坏活动
- 为胁迫或使用武力进行行动策划
- 非法行为，或对即将发生的非法行为进行煽动

以这些目的提交的内容不属于本项目范围，且可能被移除。

## 如何使用本项目

任何人都可以：

- 阅读并批评本草案
- 修改或分叉本草案
- 翻译本草案
- 用其开展教学
- 围绕其组织合法讨论
- 如果认为其中部分内容具有说服力，可围绕这些内容开展倡议

本项目的设计目的，是让他人可以在不需要单一公共领袖出面的情况下使用它。

## 维护者立场

维护者可以移除内容，或关闭试图将本项目转变为鼓吹武力、非法行为或行动协同的平台的讨论。

本仓库和网站的宗旨是进行宪法讨论与公开起草，而不是进行胁迫。

# 条款说明：竞选财务与政治平等

## 为何政治中的金钱是一个宪法问题

若大量财富集中得以主导政治传播、候选人的生存能力及获得代表权的途径，则形式上的政治平等将日益流于虚构。

正因如此，本草案将竞选财务视为民主秩序的组成部分，而非边缘性监管领域。

## 为何草案不固化某一政策模式

宪法不应试图将每一项申报门槛、公共资助比例或披露表格逐一载明。

宪法所能做的，是确立政治平等、透明度与反腐败作为宪法价值，其效力足以为对政治资金实施有实质意义的监管提供正当依据。

## 为何这对民主至关重要

其目的并非废除政治分歧或政治倡导。

而是防止代议制政府逐渐演变为一种体制——在那种体制下，进入公职与影响公共权力的途径过度依赖私人财富。

# 条款说明：总统任期限制

## 为何保留任期限制？

在总统制体系中，任期限制是防范个人统治的结构性保障。

它降低了该职位演变为以领导人为中心之政治的风险——在这种政治格局下，宪法秩序将与某一具体个人紧密相连。

## 为何在总统制下此问题更为重要

议会制通常具备更为常规的政治途径，以更换失职领导人。

总统制设有固定任期，拥有更强的个人授权，且权力高度象征性地集中于一个职位。这使得任期限制作为宪法保障措施更具正当性。

## 为何草案还包含反规避条款

若无反规避条款，任期限制可能通过继承安排或副总统轮替等手段加以规避。

因此，本草案并不将任期限制仅视为一句口号，而是力图使其具备足够的实质效力，以阻止明显的变通策略。

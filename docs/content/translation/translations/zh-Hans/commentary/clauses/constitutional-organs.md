# 条款说明：宪法机构存在的理由

## 为何不将这些职能交由普通机构承担？

某些国家职能对民主合法性而言至关重要，不能完全置于普通党派指挥链之内。

这一点对选举管理、问责执行，以及若干保护延续性的职能而言尤为如此——这些职能在宪政体制承压之际，足以决定其公信力能否维系。

## 本草案采用宪法机构的理由

其目的并非创设一个万能的第四权力分支。

其目的在于将一小组保障民主的任务，置于具备以下特征的机构之中：

- 结构上独立
- 延续性受到保护
- 难以通过普通党派阻挠手段予以解除

## 为何仍将其严格限定于较小范围

若将过多职能转移至具有隔离保护的机构，民主政府将趋于不透明与技术官僚化。

正因如此，本草案将此模式限定于一组狭窄的维系体制之职责，而非将宪法机构作为应对政治不信任的普遍解决方案。

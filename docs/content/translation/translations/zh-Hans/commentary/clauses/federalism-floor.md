# 条款说明：联邦制与民主底线

## 为何联邦制不被视为绝对原则

联邦制可以保护多元主义、制度试验与地方自治。

但若将州自治视为独立存在的宪法目的，联邦制亦可能沦为民主侵蚀、公民权利不平等及权利剥夺的庇护所。

## 为何本草案采用民主底线

本草案保留各州作为重要宪法行为主体的地位，但仅在以下内容受到保障的制度框架内方可行使：

- 共和与民主政府
- 平等公民权利
- 联邦宪法最高效力
- 基本权利与合法国家标准

其目的在于维护联邦制，同时防止其成为反民主飞地的许可证。

## 为何地方民主同样重要

本条款不仅对国家政府加以约束。

它同时承认，各州可能通过行使优先权或行政权力，在缺乏充分公共利益依据的情况下，向下侵蚀地方民主选择。本条款对联邦制的处理并非将其浪漫化为单纯的州权问题，而是将其视为一种多层次的宪法秩序。

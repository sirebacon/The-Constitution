# 条款说明：归化公民与总统职位

## 为何摒弃本土出生要求？

现行美国宪法中的本土出生公民规则在公民身份内部制造了一种等级区分。

本草案摒弃这一区分，原因在于：平等公民身份意味着，界定完整政治成员资格的应当是公民身份本身，而非血统或出生地。

## 为何此举有助于保护民主

将归化公民排除于总统职位之外，传递出这样一种信号：部分公民在宪法层面永久处于较低等级。

这与一部以平等成员资格、平等尊严及普遍政治归属为基础的宪法难以调和。

## 为何此举并非忽视安全考量

本草案并不否认，某些职位可能涉及正当的安全审查。

该问题通过范围严格受限、可接受司法审查的涉密职位规则以及一般性法律加以处理，而非依赖一种针对最高职位的、近乎世袭性质的全面排除。

# 条款说明：权力的和平移交

## 为何将和平移交载入宪法文本？

因为民主制度的保障，并不仅仅依赖于选举的举行。

其保障在于：落败者能够依法被免除权力，胜选者能够切实就任公职。若权力移交仅依赖于克制、惯例或善意，则宪政秩序将比表面上更为脆弱。

## 为何本草案将其视为核心承诺

权力的和平移交，是宪政政府与隐性政权夺取之间最为清晰的分界线。

因此，本草案通过以下机制加以保护：

- 反颠覆规则
- 选举与认证书保障措施
- 权力过渡保护规定
- 不可修改核心中的民主条款

## 为何这不仅仅是仪式

权力移交并非象征性礼仪，而是国家证明法律是否凌驾于现任者之上的时刻。

本条款之所以设立，旨在确保拒绝交出权力的行为不被仅仅视为不当举止，而是被视为对民主本身的宪法性攻击。

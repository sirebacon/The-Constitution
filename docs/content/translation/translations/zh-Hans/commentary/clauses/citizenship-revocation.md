# 条款说明：公民身份撤销与正当程序

## 本条款存在的原因

公民身份的丧失，是公权力对个人法律及政治地位所能施加的最为严厉的干预形式之一。

若政府得以通过程序简陋的手段或延迟审查来撤销公民身份，则平等成员资格将沦为有条件的、缺乏保障的权利。

## 本草案采用明确程序底线的原因

本草案规定，公民身份撤销生效前须满足宪法最低要求，包括：

- 告知
- 独立裁判机构
- 获得律师协助的权利
- 司法审查

此举并非程序上的过度苛求，而是对以下事实的确认：公民身份事关根本，不得完全交由未来的立法裁量处置。

## 本条款超越个案的重要意义

若无严格的程序保障，公民身份规则可能沦为排他性政治操弄或选择性威吓的工具。

本条款旨在明确：宪法共同体成员资格的丧失，不得以随意或政治化的方式实现。

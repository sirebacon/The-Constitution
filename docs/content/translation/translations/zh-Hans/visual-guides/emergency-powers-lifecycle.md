# 紧急权力生命周期

本可视化指南展示国家紧急状态如何开始、如何受到限制、如何失效，以及当行政部门试图扩张或规避这些限制时，宪法制度如何回应。

## 如何阅读本指南

本指南包含三条路径：

- 通常宪法路径
- 未获批准时的失效路径
- 紧急权力被拉伸超出合法边界时的滥用路径

使用路径按钮可以一次聚焦紧急制度中的一个分支。

## 本指南涵盖

- 总统宣布紧急状态
- 48小时内提交地区议会
- 30天批准期限
- 未获批准时自动失效
- 每30天续期
- 未经两院各2/3同意不得超过180天
- 司法与问责后备机制

## 本指南不做什么

本指南是导览层，不是宪法正文。

如需准确措辞和法律细节，请阅读：

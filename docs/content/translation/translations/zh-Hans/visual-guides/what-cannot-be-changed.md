# 什么不能被改变

本可视化指南专门展示宪法中的不可修宪核心：那些不能通过任何修宪路径被废除、暂停或掏空的承诺。

## 如何阅读本指南

本指南有四个部分：

- 民主结构
- 平等公民资格与权利底线
- 护栏机构
- 宪法秩序与文官统治

使用类别按钮可以一次聚焦被锁定的民主核心中的一个部分。

## 本指南涵盖

- 受到永久保护的民主最低条件
- 为什么平等公民资格被锁定
- 为什么司法审查和宪法机关受到保护
- 为什么文官控制与和平移交权力不能被修宪取消
- 为什么修宪权本身不能合法地解除这些锁定

## 本指南不做什么

本指南是导览层，不是宪法正文。

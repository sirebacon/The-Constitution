# 从选举到就职

本可视化指南提供了一条更完整的时间线，从初选和提名一直延伸到就职日，展示在每个阶段分别有哪些宪法护栏，以及当有人试图打断这条链条时会发生什么。

## 如何阅读本指南

每个阶段都是一个独立部分。使用阶段按钮可以聚焦于提名、投票日、认证或就职。每一步都附有相关宪法条文。

## 本指南涵盖

- 提名：选举委员会规则如何塑造初选、选票准入与公共资助
- 投票日：多日投票、排序选择投票，以及对借紧急状态操控选举的限制
- 认证：强制期限、选举委员会审查、欺诈调查，以及国会的事务性角色
- 就职：交接合作义务、宣誓，以及在最终交接前仍持续有效的反颠覆规则

## 本指南不做什么

本指南是导览层，不是宪法正文。

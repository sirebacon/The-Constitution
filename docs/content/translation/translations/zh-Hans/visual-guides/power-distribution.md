# 权力如何分配

本可视化指南展示宪法制度中的主要机构、民主权威从何进入，以及问责如何沿着制度结构回流。

## 如何阅读本指南

本指南围绕三层展开：

- 民主输入
- 治理机构
- 问责与审查

使用筛选按钮可聚焦于选举授权、制度制衡或问责联系。

## 本指南涵盖

- 作为民主权威来源的人民
- 众议院
- 地区议会
- 总统
- 法院
- 选举委员会
- 问责委员会

## 本指南不做什么

本指南对制度进行了简化，以便读者能够快速理解其主要架构。

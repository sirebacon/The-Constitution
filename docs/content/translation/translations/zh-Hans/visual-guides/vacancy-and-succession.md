# 职位空缺与继任

本可视化指南展示联邦职位因死亡、辞职、免职或永久丧失履职能力而出现空缺时会发生什么。

## 如何阅读本指南

每张表对应一种职位类型。使用分类按钮可聚焦于国会、总统职位、副总统职位或联邦司法系统。

## 本指南涵盖

- 众议院空缺与特别选举
- 地区议会空缺与临时任命
- 总统继任
- 副总统补缺
- 最高法院与下级法院通过资深大法官和资深法官维持连续性

## 本指南不做什么

本指南是导览层，不是宪法正文。

如需准确措辞和法律细节，请阅读：

- [第一条](articles/I-electoral-system.md) — 国会空缺与特别选举
- [第三条](articles/III-executive.md) — 总统继任与副总统空缺
- [第四条](articles/IV-judiciary.md) — 司法连续性、空缺与资深法官制度

# 权利如何得到执行

本可视化指南解释权利主张如何在宪法制度中推进：谁可以挑战违法行为、法院何时必须快速行动，以及即使在紧急状态下哪些权利仍然受到保护。

## 如何阅读本指南

本指南有三条路径：

- 通常执行
- 加速审查
- 紧急状态下的权利保护

使用路径按钮可以一次聚焦一条执行路线。

## 本指南涵盖

- 在联邦法院提起基于具体损害的权利挑战
- 对法律和行政行为的司法审查
- 对时间敏感宪法争议的加速审查
- 上级法院拖延时的后备机制
- 紧急状态下的不可减损权利
- 对权利暂停提出挑战的72小时规则

## 本指南不做什么

本指南是导览层，不是宪法正文。

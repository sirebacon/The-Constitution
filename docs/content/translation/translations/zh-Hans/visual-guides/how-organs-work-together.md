# 宪法机关如何协同运作

本可视化指南解释每个宪法机关分别做什么、它们之间如何相互联系，以及这种设计如何防止任何单一机关或官员变得居于支配地位。

## 如何阅读本指南

每张卡片对应一个机关或一类跨机关关系。使用类别按钮可以聚焦于选举委员会、问责委员会、法院与司法行为委员会，或维系整个制度的跨机关制衡。

## 本指南涵盖

- 选举委员会：它管理什么、向谁负责，以及如何受到保护
- 问责委员会：调查范围、移送触发条件，以及为什么被调查对象不能关闭它
- 法院与司法行为委员会：它们如何审查各分支行为，以及如何处理司法纪律
- 跨机关制衡：任命分离、相互审查，以及为何制度性破坏本身就是宪法违法

## 本指南不做什么

本指南是导览层，不是宪法正文。

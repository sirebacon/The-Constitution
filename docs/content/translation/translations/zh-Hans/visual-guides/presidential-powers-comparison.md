# 总统权力：今昔对比

本可视化指南比较现行美国总统职位与本草案中的总统职位，展示哪些权力仍然强大、哪些被收窄，以及新增了哪些问责限制。

## 如何阅读本指南

每一行比较一项总统权力或限制。使用分类按钮可聚焦于核心行政权力、任命与行政管理、紧急与战争权力，或问责限制。

## 本指南涵盖

- 否决权与行政命令
- 任命、代理官员与罢免
- 紧急状态声明与三军统帅限制
- 赦免与宽赦限制
- 公民罢免、弹劾以及类别性豁免的终结

## 本指南不做什么

本指南是导览层，不是宪法正文。

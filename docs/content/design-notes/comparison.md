# Comparison: Current vs. Revised Constitution

A side-by-side reference showing what changed and why.

---

| Area | Current US Constitution | Revised Constitution |
|------|------------------------|---------------------|
| **Voting rights** | Not explicitly guaranteed; inferred and protected via amendments | Explicit constitutional right; automatic registration |
| **Presidential election** | Electoral College | Direct national popular vote with RCV |
| **Voting system** | Not specified (states use FPTP by default) | Ranked Choice Voting mandated federally |
| **Congressional districts** | Single-member, winner-take-all | Multi-member proportional districts |
| **Gerrymandering** | No protection; left to states/courts | Independent algorithmic redistricting mandated |
| **Senate** | 2 senators per state regardless of population | Regional Assembly, population-weighted |
| **Filibuster** | Not in constitution; Senate rule | Prohibited — majority rule restored |
| **Emergency powers** | No time limit (National Emergencies Act is statute, not constitution) | 30-day limit, Congress must renew |
| **Pardon power** | Broad, no explicit limits | No self-pardon; no pardoning co-conspirators |
| **War powers** | Ambiguous; Congress declares but President commands | Congressional authorization required |
| **Supreme Court size** | 9 justices (set by statute) | 15 justices (set by constitution) |
| **Justice tenure** | Life tenure | 15-year staggered terms |
| **Justice appointments** | President + Senate majority | President + nonpartisan commission |
| **Judicial ethics** | No constitutional requirement | Mandatory ethics code with enforcement |
| **Privacy** | Inferred (4th Amendment penumbra) | Explicit constitutional right |
| **Equal rights** | Equal protection clause (14th); ERA never ratified | Explicit equality across race, sex, gender, orientation, disability |
| **Affirmative rights** | None | Healthcare floor, education, housing security |
| **Environmental rights** | None | Right to clean air, water, stable climate |
| **Digital rights** | None | Explicit data privacy and algorithmic rights |
| **Gun rights** | "Well-regulated militia" — contested | Individual right, subject to reasonable regulation |
| **Anti-coup provisions** | None | Permanent disqualification + criminal liability |
| **Campaign finance** | Largely struck down by courts | Constitutionalized limits and disclosure |
| **Revolving door** | Statutory only, easily changed | 5-year restriction constitutionalized |
| **Media pluralism** | No constitutional provision | Ownership concentration limits |
| **Amendment process** | Single track: 2/3 + 3/4 states (very hard) | Two tracks: structural (hard) + rights expansion (moderate) |
| **Unamendable core** | None | Voting rights, anti-coup, judicial independence |
| **Territories/D.C.** | No statehood; non-voting representation | Defined path to statehood; full representation |

---

## What Was Preserved

- Separation of powers (executive, legislative, judicial)
- Bicameralism (reformed, not eliminated)
- Federalism (states retain reserved powers)
- All classic civil liberties (speech, religion, press, assembly, due process, etc.)
- Presidential system (not replaced with parliamentary)
- Federal supremacy over state law on enumerated matters
- Checks and balances between branches

# How Power Is Distributed

This visual guide shows the main institutions in the constitutional system, where democratic authority enters, and how accountability runs back through the structure.

## How To Read This Guide

The guide is organized around three layers:

- democratic input
- governing institutions
- accountability and review

Use the filter buttons to focus on electoral authority, institutional checks, or accountability links.

## What This Guide Covers

- the People as the democratic source of authority
- House of Representatives
- Regional Assembly
- President
- Courts
- Electoral Commission
- Accountability Commission

## What This Guide Does Not Do

This guide simplifies the system so readers can understand the main architecture quickly.

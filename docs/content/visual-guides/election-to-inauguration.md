# From Election To Inauguration

This visual guide is a step-by-step timeline from primaries through inauguration day, showing where the constitutional guardrails are at each phase and what happens when someone tries to break the chain.

## How To Read This Guide

Each phase is a separate section. Use the phase buttons to focus on nomination, election day, certification, or inauguration. Every step includes the relevant constitutional provision.

## What This Guide Covers

- Nomination: how EC rules govern primaries, ballot access, and public financing
- Election Day: multi-day polling, ranked-choice voting, and emergency manipulation limits
- Certification: mandatory deadlines, EC review, ACC fraud investigations, and the ministerial congressional role
- Inauguration: transition cooperation duties, the oath, and anti-subversion rules that survive the handover

## What This Guide Does Not Do

This guide is an orientation layer, not the constitutional text itself.

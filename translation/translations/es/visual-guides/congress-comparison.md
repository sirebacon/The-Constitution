# El Congreso: Antes Y Ahora

Esta guía visual muestra cómo se compara la legislatura de este borrador con el Congreso actual de Estados Unidos: qué cambió, qué se mantuvo y por qué.

## Cómo Leer Esta Guía

Cada fila compara una característica estructural. Usa los botones de categoría para centrarte en composición, poderes, rendición de cuentas o diferencias de procedimiento.

## Qué Cubre Esta Guía

- composición de las cámaras y representación
- duración de los mandatos y ciclos electorales
- poderes exclusivos de cada cámara
- reglas de rendición de cuentas y remoción
- reglas de procedimiento, incluido el filibusterismo
- poderes de confirmación y tratados

## Qué No Hace Esta Guía

Esta guía es una capa de orientación, no el texto constitucional en sí.

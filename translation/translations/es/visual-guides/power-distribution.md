# Cómo Se Distribuye El Poder

Esta guía visual muestra las instituciones principales del sistema constitucional, por dónde entra la autoridad democrática y cómo la rendición de cuentas vuelve a recorrer la estructura.

## Cómo Leer Esta Guía

La guía se organiza en torno a tres capas:

- entrada democrática
- instituciones de gobierno
- rendición de cuentas y revisión

Usa los botones de filtro para centrarte en la autoridad electoral, los controles institucionales o los vínculos de rendición de cuentas.

## Qué Cubre Esta Guía

- el Pueblo como fuente democrática de autoridad
- Cámara de Representantes
- Asamblea Regional
- Presidente
- Tribunales
- Comisión Electoral
- Comisión de Rendición de Cuentas

## Qué No Hace Esta Guía

Esta guía simplifica el sistema para que los lectores puedan entender rápidamente la arquitectura principal.

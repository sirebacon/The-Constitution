# Nota de Cláusula: El Federalismo y el Umbral Democrático

## Por Qué el Federalismo No Se Trata Como Absoluto

El federalismo puede proteger el pluralismo, la experimentación y el autogobierno local.

Sin embargo, también puede convertirse en refugio para la erosión democrática, la ciudadanía desigual y la denegación de derechos si la autonomía estatal se trata como un fin constitucional independiente.

## Por Qué Este Proyecto Utiliza un Umbral Democrático

Este proyecto preserva a los estados como actores constitucionales importantes, pero únicamente dentro de un sistema que protege:

- el gobierno republicano y democrático
- la ciudadanía igual
- la supremacía constitucional federal
- los derechos fundamentales y las normas nacionales legítimas

El propósito es preservar el federalismo sin permitir que se convierta en licencia para la existencia de enclaves antidemocráticos.

## Por Qué la Democracia Local También Importa

El artículo no solo limita al gobierno nacional.

También reconoce que los estados pueden extralimitarse hacia abajo, utilizando la preemción o el poder administrativo para bloquear decisiones democráticas locales sin un interés público suficiente. El federalismo, en este contexto, no se idealiza como una cuestión exclusiva de derechos estatales. Se trata como un orden constitucional estratificado.

# Por Qué el Comentario Es Independiente

## La Elección Fundamental

Este proyecto utiliza dos capas:

- el texto constitucional para las normas vinculantes
- el comentario para la explicación y la justificación

Esa distinción es deliberada.

## ¿Por Qué No Incluir la Justificación en la Constitución?

Porque el texto constitucional debe cumplir únicamente algunas funciones con precisión:

- distribuir la autoridad
- imponer deberes
- establecer límites
- definir remedios y mecanismos de salvaguarda

Una vez que la redacción del comentario se mezcla con el texto del artículo, el documento comienza a leerse como un híbrido de constitución, artículo de revista jurídica y manual de implementación.

## Lo Que Protege la Separación

Mantener el comentario de forma independiente protege:

- la estabilidad de las citas
- la legibilidad
- la claridad en la traducción
- el mantenimiento futuro
- una distinción clara entre el derecho y la explicación

## Lo Que el Comentario Aporta de Todos Modos

El comentario independiente permite igualmente que el proyecto explique decisiones controvertidas o poco familiares, entre ellas:

- el núcleo inalterable
- los órganos constitucionales
- los poderes de emergencia de alcance restringido
- la decisión de mantener un sistema presidencial más seguro en lugar de adoptar el parlamentarismo
- los mecanismos de antidemora y antisubversión

Ese es el equilibrio adecuado para un proyecto constitucional de carácter público: derecho claro, razonamiento visible.

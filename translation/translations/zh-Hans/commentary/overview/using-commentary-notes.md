# 关于注释说明的使用

## 用途

本站有意将宪法文本与说明性材料分开呈现。

宪法是具有效力的文本。注释说明则解释特定条款存在的缘由、其所针对的问题，以及影响起草选择的各种权衡取舍。

## 阅读方法

如需了解规则本身，请查阅宪法文本。

如需了解以下内容，请查阅注释说明：

- 某一条款背后的民主问题
- 该条款旨在防止的滥权模式
- 本草案选择某一方案而非其他方案的理由
- 某些细节留待普通立法而非宪法化处理的原因

## 注释的性质界定

注释不具有约束力。

注释不属于宪法的组成部分，亦无意凌驾于条款文本之上或对其作出隐性修改。注释仅为读者、审阅者及译者提供一个说明性层次。

## 重要意义

宪法应保持作为法律文本的可读性。

若将相关理由、历史例证及起草争议直接嵌入条款文本，文件的引用、维护将愈加困难，亦将失去宪法应有的文本风格。独立的注释层既保持了法律文本的简洁，又使各项选择清晰可辨。

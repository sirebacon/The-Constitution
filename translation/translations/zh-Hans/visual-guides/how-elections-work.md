# 选举如何运作

本可视化指南解释在本宪法下联邦选举有哪些不同：投票方式如何运作、登记如何进行、哪些保护措施防止操控，以及选民如何罢免官员。

## 如何阅读本指南

每张卡片解释选举制度中的一个特征。使用分类按钮可聚焦于投票方式、投票渠道与登记、诚信保护或罢免机制。

## 本指南涵盖

- 众议院采用排序投票和多成员选区
- 地区议会采用即时决选投票
- 自动选民登记
- 防止借紧急状态操纵选举日期的保护
- 带有反杰利蝾螈规则的独立重划选区
- 选民可罢免众议院和地区议会成员

本指南是导览层，不是宪法正文。

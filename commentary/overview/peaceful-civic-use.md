# Peaceful Civic Use

## Project Purpose

This project exists for peaceful civic discussion, constitutional drafting, education, translation, critique, simulation, and democratic advocacy.

It is a public constitutional reference project. It is not a militia, an operational organization, or a vehicle for coercion or force.

## What The Project Does Not Endorse

This project does not endorse:

- violence
- threats
- sabotage
- operational planning for coercion or force
- unlawful action or encouragement of imminent unlawful action

Material submitted for those purposes is outside the project and may be removed.

## How To Use The Project

People are free to:

- read and criticize the draft
- revise or fork it
- translate it
- teach from it
- organize lawful discussion around it
- campaign on parts of it if they find them persuasive

The project is meant to be usable by others without requiring a single public leader to stand in front of it.

## Maintainer Position

Maintainers may remove content or close discussions that try to turn the project into advocacy of force, unlawful action, or operational coordination.

The purpose of the repository and website is constitutional discussion and public drafting, not coercion.

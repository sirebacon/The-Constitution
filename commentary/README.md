# Commentary Structure

This directory holds non-binding explanatory material for the website.

## Layout

- `overview/`
  site-level commentary pages explaining how to read the project and why certain presentation choices were made
- `articles/`
  article-level notes keyed to the matching file name in `articles/`

## Rule

The constitutional text in `articles/` and `preamble.md` remains the operative source text.

Files in this directory are explanatory only. They are meant to justify design choices, explain historical or institutional concerns, and make the project easier to read without mixing argument into the constitutional language itself.

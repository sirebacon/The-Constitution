# Clause Note: Why Constitutional Organs Exist

## Why Not Leave These Functions To Ordinary Agencies?

Some state functions are too central to democratic legitimacy to be left entirely inside the ordinary partisan chain of command.

This is especially true for election administration, accountability enforcement, and a few continuity-protecting functions that can decide whether the constitutional system remains credible under stress.

## Why This Draft Uses Constitutional Organs

The point is not to create an all-purpose fourth branch.

The point is to place a small set of democracy-preserving tasks in institutions that are:

- structurally independent
- continuity-protected
- harder to disable through ordinary partisan obstruction

## Why This Is Still Kept Narrow

If too many functions are moved into insulated bodies, democratic government becomes opaque and technocratic.

That is why the draft limits this model to a narrow set of system-preserving duties rather than using Constitutional Organs as a general answer to distrust in politics.

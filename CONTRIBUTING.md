# Contributing

This repository is an open constitutional drafting project for peaceful civic use.

Contributions are welcome if they help improve:

- constitutional text
- commentary and explanation
- simulations and testing
- translations
- site usability and accessibility
- documentation and research support

## Project Boundaries

This project exists for peaceful civic discussion, constitutional drafting, education, translation, critique, simulation, and democratic advocacy.

It does not endorse:

- violence
- threats
- sabotage
- unlawful action
- operational planning for coercion or force

Contributions that advocate, plan, or normalize those uses are outside the project and may be rejected or removed.

## Contribution Norms

Contributors should aim to keep the project:

- readable
- nonviolent
- constitutionally serious
- open to criticism and revision
- usable by people who do not share the same ideology

When proposing changes:

- keep constitutional text separate from commentary where possible
- explain the problem the change is solving
- avoid turning the Constitution into a statutory code unless clearly necessary
- preserve citation stability and clarity where practical

## Legal And Interpretive Caution

Nothing in this repository should be read as encouraging imminent unlawful conduct, conspiracy, or the use of force against persons or institutions.

This is a public drafting and discussion project. It is not a militia, an operational organization, or a vehicle for unlawful action.

# Constitution Scorecard

Use this document to track the quality of the draft over time. The goal is not just to rate the text, but to identify where improvement work should go next.

---

## Scoring Scale

Score each area from **1 to 10**.

- **10** — Excellent; coherent, enforceable, and near-final
- **8–9** — Strong; only targeted revisions needed
- **6–7** — Good foundation; meaningful revision still needed
- **4–5** — Conceptually promising but underdeveloped or unstable
- **1–3** — Major redesign needed

---

## Evaluation Criteria

Score each section on these dimensions:

| Criterion | What to Ask |
|----------|-------------|
| **Clarity** | Is the rule understandable on first read? |
| **Internal Consistency** | Does it fit cleanly with the rest of the Constitution? |
| **Enforceability** | Can a court, commission, or political actor actually apply it? |
| **Democratic Legitimacy** | Does it preserve accountability and majority rule with minority protection? |
| **Abuse Resistance** | Does it close obvious bad-faith or authoritarian loopholes? |
| **Draft Discipline** | Is it concise enough to function as constitutional text rather than statutory code? |

Suggested scoring method:

1. Give each criterion a score from 1 to 10.
2. Average them for a section score.
3. Round to the nearest half-point if needed.
4. Record the main reason for the score and the next fix to improve it.

## Review Checklist

Before raising a score, ask:

- Did the revision make the text clearer?
- Did it reduce contradiction risk?
- Did it improve enforceability in practice?
- Did it preserve democratic legitimacy?
- Did it remove detail that belongs in statute instead?
- Did it avoid introducing new cross-reference drift?

# Emergency Powers Lifecycle

This visual guide shows how a national emergency begins, how it is constrained, how it lapses, and how the constitutional system responds if the executive tries to stretch or evade those limits.

## How To Read This Guide

The guide has three paths:

- ordinary constitutional path
- lapse path when approval is not granted
- abuse path when emergency power is stretched beyond lawful limits

Use the path buttons to focus on one branch of the emergency system at a time.

## What This Guide Covers

- presidential declaration
- 48-hour submission to the Regional Assembly
- 30-day approval deadline
- automatic lapse if approval is not granted
- 30-day renewals
- 180-day maximum without a 2/3 vote of both chambers
- judicial and accountability backstops

This guide is an orientation layer, not the constitutional text itself.
